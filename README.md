# Layer_01: Not Found


A custom 404 error page inspired by the 1998  classic anime **Serial Experiments Lain**.


##  The Concept

In the series finale, Lain Iwakura resets reality and erases herself from the collective memory of the world. This 404 page treats the user's request as a search through "The Wired" for a subject that no longer exists in any memory address.
